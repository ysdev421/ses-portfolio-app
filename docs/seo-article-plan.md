# SEO記事計画（SESキャリア記録）

## 方針
- 目的: 未ログインユーザーの検索流入を獲得し、`/signup` へ送客する
- 想定読者: SESエンジニア（1〜5年目）、案件選び・面談・職務経歴書に悩む層
- 基本ルール: 1記事1検索意図、見出しに主要キーワードを含める、末尾CTAは「無料登録」

## 内部リンク設計（ハブ&スポーク）
- ハブページ:
  - `/news`（最新情報）
  - 将来追加: `/guides`（記事一覧）
- クラスター:
  - 書類系: #1 #2 #7
  - 面談系: #3 #10
  - 案件選定/単価: #4 #5 #6 #9
  - キャリア設計: #8
- リンクルール:
  - 各記事に「関連記事」3本を固定表示
  - 各記事末尾に `/signup` CTA を設置
  - 記事冒頭にパンくず（`ホーム > ガイド > 記事名`）
